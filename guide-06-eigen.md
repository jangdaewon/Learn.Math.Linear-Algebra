# 단원 6. 고윳값 · 고유벡터 · 대각화

## 학습 목표

* 고윳값·고유벡터의 정의를 기하적으로 이해하고 계산 방법을 익힌다.
* 특성방정식(det(A−λI)=0)을 이용해 고윳값을 찾고, 대응 고유벡터를 구한다.
* 고유분해(Eigendecomposition)·대각화가 계산을 단순화하는 이유를 체득한다.
* 대각화 가능 조건(고유벡터의 개수·기저 형성)을 사례로 확인한다.

## 핵심 정의와 직관

| 개념     | 직관 지침                 | 수식·조건                     |
| ------ | --------------------- | ------------------------- |
| 고유방정식  | 변환 후 **방향 불변**, 길이 λ배 | A v = λ v, v≠0            |
| 특성방정식  | ‘방향 불변’ 찾기 ↔ 행렬식 0    | det(A − λI) = 0           |
| 고유벡터·값 | 고유축 + 신축/압축 계수        | λ >1 확대, 0<λ<1 축소, λ<0 반전 |
| 대각화    | 고유축 좌표계로 변환 → 계산 단순   | A = PDP⁻¹ with D diag(λᵢ) |
| 대각화 조건 | n개 독립 고유벡터 ⇔ 대각화 가능   | dim eigenspace 합 = n      |

## 이해 검증 과제

| #    | 문제                                                                         | 풀이 공간 |
| ---- | -------------------------------------------------------------------------- | ----- |
| 1    | 행렬 A = (\begin{bmatrix}2&0\0&3\end{bmatrix}) 의 고윳값·고유벡터를 구하고 GeoGebra로 시각화 |       |
| 2    | 90° 회전 행렬 R 의 고윳값이 실수 영역에 존재하지 않음을 설명 (복소수 힌트)                             |       |
| 3    | A = (\begin{bmatrix}1&1\0&1\end{bmatrix}) 의 고윳값·고유벡터를 찾고 대각화 가능 여부 판단      |       |
| 4 ★  | 3×3 대칭 행렬 B = [[4,1,0],[1,4,0],[0,0,2]] 의 고유분해 수행·P, D 구하기                 |       |
| 5 ★★ | 고유벡터가 2개뿐인 3×3 행렬의 구체적 예를 만들고 ‘대각화 불가’ 이유를 서술                              |       |

> ★ = 심화 계산, ★★ = 개념+창의 응용
