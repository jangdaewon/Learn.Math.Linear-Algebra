# 학습 노트 작성 가이드

> **목적**
> 이 가이드는 8주 학습 로드맵의 **세부 항목**(🄰 개념, 🄱 예시, 🄲 문제 등)을 개인 학습 노트로 정리할 때 일관된 형식을 제공한다. 핵심 원칙은 **시각적·직관적 이해**와 **단순·인위적 문제로 개념 검증**이다.

```

## 2) 직관 메모
- 비유·애니메이션 링크·GeoGebra 스크립트 경로

## 3) 공식 정리
- 핵심 수식 1~3개 (Latex)
- 각 수식당 의미를 짧은 주석으로 달기

## 4) 연습문제 & 풀이
| # | 문제 (★ 난도 표기) | 한줄 해설 |
|---|-------------------|-----------|
| 1 | … | … |
| 2 | … | … |

## 5) 셀프 체크 리스트
- [ ] …
- [ ] …

## 6) 오답 & 회고
- 다시 틀린 개념 요약 · 수정 그림
```

### 작성 요령

| 섹션           | 지침                                          | 예시                             |
| ------------ | ------------------------------------------- | ------------------------------ |
| ① **개념 스냅샷** | 1 컷 그림 + 1 문장. 글보다 **그림 우선**.               | 평행사변형 그림 + “벡터 덧셈 = 화살표 이어붙이기” |
| ② **직관 메모**  | 비유·실험·링크. **시각·감각적 언어** 사용.                 | “스프링을 늘이는 느낌 → 스칼라배”           |
| ③ **공식 정리**  | 최대 3개. **항상 주석** 달기.                        | (‖v‖ = √(x²+y²)) ← 벡터 길이       |
| ④ **연습문제**   | 난도 ★~★★. **계산 5줄 이하**. 인위적 숫자(정수·0,1,2) 사용. | (2,1)+(−1,4)=?                 |
| ⑤ **체크 리스트** | “입으로 설명 가능?” 형태. **3~5개**.                  | □ 덧셈 평행사변형 말로?                 |
| ⑥ **오답 회고**  | 틀린 이유 + 수정그림. **1페이지 제한**.                  | 경로 헷갈림 화살표 재그림                 |

## 2. 시각 자료 제작 지침

1. **GeoGebra**

    * 2D/3D 화살표, 격자 변형, 투영 등 시뮬레이션 저장.
    * 파일명: `weekX-topicY.ggb`.
2. **Matplotlib/Python** (선택)

    * 간단한 점·화살표 그리기용 스크립트.
3. **GIF 또는 PNG**

    * 정지 1컷 또는 3~5초 루프 GIF.
4. **링크 관리**

    * 노트 상단에 파일·영상 링크 표.

---

## 3. 연습문제 출제 원칙

| 항목     | 내용                                    |
| ------ | ------------------------------------- |
| **목적** | “개념 ✓/✗” 판단. 현실 맥락 필요 없음.             |
| **형식** | 정수·단순 분수 사용. 2D·3D 벡터는 (x,y[,z]) ≤ 3. |
| **난도** | ★ (직접적) / ★★ (한 단계 응용). ★★★ 이상 금지.    |
| **해설** | 정답 + 한줄 이유. 상세 증명은 별도 풀이노트에.          |

## 5. 버전 관리 & 복습 루프
