## 핵심 정의와 직관

| 개념          | 직관 지침                       | 수식·조건                   |
| ----------- | --------------------------- | ----------------------- |
| 행렬곱 **AB**  | 변환 **B** 후 **A** 적용 (연속 이동) | (AB)x = A(Bx)           |
| 단위행렬 **I**  | ‘아무 일도 안 일어남’ 변환            | Ix = x                  |
| 역행렬 **A⁻¹** | 변환을 **원상복구**                | A⁻¹A = AA⁻¹ = I         |
| 선형 변환       | 원점 고정 + 직선 보존               | T(u+v)=Tu+Tv, T(cv)=cTv |
| 교환 실패       | 회전·스케일 교환 시 다른 격자           | 일반적으론 AB≠BA             |
