## 핵심 정의와 직관

| 개념   | 직관 지침             | 수식                      |
| ---- | ----------------- | ----------------------- |
| 벡터   | 시작점→끝점 화살표, “이동”  | **v = (v₁, …, vₙ)**     |
| 덧셈   | 화살표 이어 붙이기, 평행사변형 | **u + v**               |
| 스칼라배 | 화살표 늘이기/줄이기       | **c u**                 |
| 크기   | 피타고라스 길이          | ‖**v**‖ = √(Σ vᵢ²)      |
| 단위벡터 | 방향만 남기기           | **û** = **v** / ‖**v**‖ |
