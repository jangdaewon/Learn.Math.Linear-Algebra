# 단원 7. 내적 · 직교 · 그래므-슈미트

## 학습 목표

* 내적(inner product) 정의와 ‘각도·길이’ 관계를 직관적으로 이해한다.
* 직교(orthogonality)·직교 기저의 장점을 시각적으로 체험한다.
* 그래므-슈미트(Gram–Schmidt) 과정을 단계별로 수행해 본다.
* 직교 투영(orthogonal projection)의 개념을 화살표 그림으로 설명할 수 있다.

## 핵심 정의와 직관

| 개념         | 직관 지침                         | 수식·조건                      |
| ---------- | ----------------------------- | -------------------------- |
| 내적 **u·v** | **u** 를 **v** 위로 투영한 길이 × ‖v‖ | u·v = ‖u‖‖v‖cosθ           |
| 직교         | 내적이 0 → 90°                   | u·v = 0                    |
| 직교 기저      | 좌표 축이 서로 ‘완전 독립’              | ‖eᵢ‖=1, eᵢ·eⱼ=0 (i≠j)      |
| 그래므-슈미트    | 기존 벡터를 차례로 투영·빼서 직교 집합 생성     | vᵢ' = vᵢ − Σ proj_{eⱼ}(vᵢ) |
| 직교 투영      | 벡터를 부분공간 위에 ‘떨어뜨리기’           | proj_W(u) = Σ (u·eᵢ) eᵢ    |

## 이해 검증 과제

| #    | 문제                                                                         | 풀이 공간 |
| ---- | -------------------------------------------------------------------------- | ----- |
| 1    | 벡터 **a = (2,1)**, **b = (−1,2)** 의 내적과 각도를 계산·직교 여부 확인                     |       |
| 2    | R³ 벡터 집합 { (1,1,0), (−1,1,0), (0,0,2) } 을 그래므-슈미트 1단계 수행 후 e₁ 구하기          |       |
| 3    | 2D 벡터 **u = (3,4)** 를 직교 기저 { (1,0), (0,1) } 위에 투영할 때 결과가 왜 **u** 와 같은지 설명 |       |
| 4 ★  | R³ 부분공간 W = Span{ (1,0,1), (0,1,1) } 에서 **v = (2,3,4)** 의 직교 투영을 계산        |       |
| 5 ★★ | 그래므-슈미트가 실패(0 벡터 발생)할 수 있는 경우 예시를 들고 이유를 설명                                |       |

> ★ = 심화 계산, ★★ = 개념 응용·반례 찾기
