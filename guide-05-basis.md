# 단원 5. 벡터 공간 · 기저 · 차원

## 학습 목표

* 벡터 공간과 부분공간 공리를 이해하고 예시를 분류할 수 있다.
* 기저(Basis)의 의미를 직관적으로 파악하고 실제로 구성해 본다.
* 차원(Dimension)의 개념을 기저와 연결해 설명할 수 있다.
* 좌표계 변경(Change of Basis)이 벡터 표현을 어떻게 바꾸는지 시각화로 확인한다.

## 핵심 정의와 직관

| 개념    | 직관 지침                       | 공식·조건        |   |   |
| ----- | --------------------------- | ------------ | - | - |
| 벡터 공간 | ‘덧셈·스칼라배가 가능한 세계’           | 8개 공리(요약 암기) |   |   |
| 부분공간  | 0 포함, 덧셈·스칼라배 닫힘            | W ⊆ V, 0 ∈ W |   |   |
| 기저    | ‘새 축’ · 독립+Span 동시에 만족      | 독립 & Span(V) |   |   |
| 차원    | 최소 기저 벡터 수                  | dim V =      | B |   |
| 좌표 변환 | 기저 바뀌면 숫자표시가 바뀜, 벡터 자체는 그대로 | x = PB [x]_B |   |   |

## 이해 검증 과제

| #    | 문제                                                                      | 풀이 공간 |
| ---- | ----------------------------------------------------------------------- | ----- |
| 1    | 집합 S = { (2, 1), (4, 2) } 가 R² 부분공간인지 설명                                |       |
| 2    | 다항식 공간 P₂ = { a + bx + cx² } 의 차원과 한 가지 기저를 제시                          |       |
| 3    | 행렬 A = [[1,2,3],[2,4,6],[1,1,1]] 의 행공간(Row Space) 기저를 가우스 소거로 구하고 차원 기재 |       |
| 4 ★  | 벡터 v = (3, 2) 를 기저 B = { (1, 1), (−1, 2) } 좌표계로 표현하기 → [v]_B 계산         |       |
| 5 ★★ | R³ 부분공간 W = Span{ (1,0,1), (0,1,1) } 의 직교 기저를 그래므-슈미트로 구하라              |       |

> ★ = 개념 응용, ★★ = 그래므-슈미트 포함 복합 문제
