# 단원 8. 대칭 행렬 · 분해 · 최소제곱 & 종합 복습

## 학습 목표

* 대칭(Symmetric) 행렬의 고유분해(Eigendecomposition)를 계산하고 기하적으로 이해한다.
* QR · LU 등의 기본 행렬 분해 아이디어를 맛본다.
* 최소제곱(Least Squares) 문제를 직교 투영 관점으로 해석하고 해를 구한다.
* 앞선 1–7단원 핵심 개념을 한눈에 연결하는 종합 개념 지도를 작성한다.

## 핵심 정의와 직관

| 개념           | 직관 지침                       | 수식·조건                         |
| ------------ | --------------------------- | ----------------------------- |
| 대칭 행렬        | ‘좌우 거울’ → 항상 직교 고유벡터        | Aᵀ = A, A = Q Λ Qᵀ            |
| 고유분해         | 직교 행렬 Q (열=고유벡터) + 대각 Λ(λᵢ) | A = QΛQᵀ                      |
| QR 분해        | ‘직교 + 상삼각’ 분해, 선형 시스템 안정 해법 | A = QR                        |
| LU 분해        | ‘하·상삼각’ 분해, 가우스 소거 기록       | A = LU                        |
| 최소제곱         | 해가 없을 때 최단 잔차 해, 투영 해       | (\hat x = (A^T A)^{-1} A^T b) |
| 잔차(residual) | b − A \hat x, W⊥ 방향 화살표     |                               |
