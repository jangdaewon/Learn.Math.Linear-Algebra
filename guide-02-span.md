## 핵심 정의와 직관

| 개념   | 직관 지침                     | 공식·조건                           |
| ---- | ------------------------- | ------------------------------- |
| 선형결합 | 여러 화살표를 늘이고 겹쳐 **종점** 모으기 | **c₁v₁ + … + cₖvₖ**             |
| Span | 가능한 종점 집합 → ‘채워지는 영역’     | **Span{v₁,…,vₖ}**               |
| 선형독립 | 벡터 중 하나도 나머지 결합으로 만들 수 없음 | **cᵢ = 0 (모든 i) → Σ cᵢ vᵢ = 0** |

## 이해 검증 과제

| #   | 문제                                                      | 풀이 공간 |
| --- | ------------------------------------------------------- | ----- |
| 1   | **v₁ = (1,2), v₂ = (3,6)** 의 Span이 선인지 면인지 설명           |       |
| 2   | **(1,0,0), (0,1,0), (0,0,1)** 의 Span이 R³ 전부임을 그림과 말로 증명 |       |
| 3   | **(2,1)** 를 **(1,3), (0,2)** 의 선형결합으로 표현 가능한지 계수 찾기     |       |
| 4 ★ | 3×3 행렬 행벡터가 선형독립인 경우 랭크가 얼마인지 설명                        |       |
| 5 ★ | **u = (1,1,1), v = (1,2,3)** 의 독립성 확인 (가우스 소거 스텝 기록)    |       |

> ★ 표시는 난도 상승.
